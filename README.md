# Microscope Specimen Size Calculator

A Streamlit web app that calculates the real-life size of a microscope specimen from image data.

## Usage

Enter:
- Magnification power of the lens
- Size of the specimen as seen in the image
- Unit of the image measurement
- Desired unit for the real-life output

The app computes: **real size = image size ÷ magnification**

## Run locally

```bash
pip install streamlit
streamlit run app.py
```
